# Movie Recommendation System

## Overview
This project is an interactive web application designed to recommend movies based on user preferences. It was developed as part of a Data Analyst training program.

The application helps users discover relevant movies by analyzing their preferences and behavior using data-driven techniques.

---

## Features

- Personalized Recommendations  
  Suggests movies based on user preferences and interaction patterns.

- Data Integration  
  Combines internal datasets with external APIs such as TMDb and IMDb.

- Machine Learning Models  
  Uses algorithms to analyze data and generate relevant recommendations.

- Interactive Web Application  
  Built with Streamlit to provide a simple and user-friendly interface.

- Performance Dashboard  
  Includes a Power BI dashboard to visualize usage metrics and application performance.

---

## Tech Stack

- Python  
- Machine Learning  
- APIs (TMDb, IMDb)  
- Streamlit  
- Databases  
- Power BI  

---

## How It Works

1. The user interacts with the Streamlit application.  
2. Preferences and inputs are collected.  
3. Data is processed using machine learning algorithms.  
4. The system generates personalized movie recommendations.  
5. Usage and performance data can be visualized through Power BI.
